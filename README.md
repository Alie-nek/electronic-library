 ### Важное замечание о хранении файлов

На бесплатном тарифе Render используется эфемерная файловая система — 
файлы обложек не сохраняются между перезапусками сервера. 

В связи с этим, обложки, загруженные до перезапуска сервера, могут не отображаться. 
Это проблема не кода, а бесплатного тарифа Render.

**Как проверить работоспособность функционала:**
Достаточно добавить новую книгу через аккаунт администратора — обложка сохранится 
и будет отображаться корректно. Весь остальной функционал (добавление книг, рецензии, 
статистика, экспорт CSV) работает без ошибок.
